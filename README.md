# Image Generator

AI-powered image generation application built with Next.js and Replicate API.

## Features

- AI Image Generation
- Real-time image creation
- User authentication
- Image history
- Download generated images

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Firebase
- Replicate API
